# 자동차 경주

## 기능목록

- [] 자동차 이름 입력 받기
- [] 예외처리
  - [] 이름을 `,` 로 구분 안 한 경우
  - [] `,` 가 중복된 경우
  - [] 이름이 5자 초과한 경우
- [] 실행할 횟수 입력 받기
- [] 예외처리
  - [] 숫자가 아닌 입력을 한 경우
- [] 실행결과 출력하기
- [] 최종/ 단독 우승자 출력하기
- [] 예외상황시 종료

## 주의할 점

- 들여쓰기 2까지만 가능하다 (함수 분리)
- 커밋 컨벤션
  - feat (feature)
  - fix (bug fix)
  - docs (documentation)
  - style (formatting, missing semi colons, …)
  - refactor
  - test (when adding missing tests)
  - chore (maintain)
